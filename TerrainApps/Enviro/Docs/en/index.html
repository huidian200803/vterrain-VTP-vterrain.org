VTP Enviro Documentation

Enviro is the VTP runtime environment program.  It provides an interactive, realtime 3D navigation of your virtual terrain.  There are two primary view modes: Earth View and Terrain View.  At startup, you chose which view you want to start with, and you can switch at any time.
